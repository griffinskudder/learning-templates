How to set up your Interactive Image

  1. Background image: Place your image file in the same folder as this HTML file, then change src="placeholder.svg" on the <img class="bg-image"> tag to your filename, e.g. src="my-image.jpg".
  2. Add a button: Inside <div class="image-wrap">, copy any <button class="pin-btn"> block, give it a unique data-popup value, and position it with top / left percentages.
  3. Add a popup: Copy any <div class="popup"> block, match its id to the button's data-popup, and fill in your title & text.
  4. Change colours & fonts: Edit the CSS variables in the :root { } block at the top of the <style> section.
  5. Delete this instructions box when you are done.
