How to customise this map

  1. Change the title: Edit the <h1> and .subtitle text below.
  2. Change colours / font: Edit the CSS variables in the :root { } block at the top of the <style> tag.
  3. Add a marker: Copy one of the existing <div class="marker"> elements inside .map-area and paste it below the others. Change top and left (percentages from top-left corner of the map) to position it, then update data-title and data-desc.
  4. Remove a marker: Delete its <div class="marker"> line.
  5. Delete these instructions: Remove the entire <div class="instructions"> block before submitting.

Tip: To find the right top / left values, open this file in a browser, right-click the map, choose Inspect, then hover over the map area to see pixel coordinates — divide by the map's total width/height to get percentages.
