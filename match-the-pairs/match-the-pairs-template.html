Step 1 - Title: Find var TITLE and var SUBTITLE near the top of the <script>. Change them to your topic.

Step 2 - Pairs: Find the var pairs array. Replace the examples with your own terms and definitions. Each entry looks like:

{ term: "Your Term", definition: "Your definition here.", image: "" }

Step 3 - Images (optional): To show an image instead of (or alongside) a definition, put the image filename in the image field, e.g. image: "erosion.jpg". The image file must be in the same folder as this HTML file. Leave it as "" for text-only pairs.

Step 4 - Colours: Change the CSS variables in the CUSTOMISATION ZONE at the top of the <style> block.
