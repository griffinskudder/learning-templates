How to set up your quiz

  1. Change <title> at the top and the <h1> below to your quiz name.
  2. Edit the text inside class="question-text" to write your question.
  3. Edit the four <button> labels to your four answer choices.
  4. On the correct button only, add data-correct="true".
  5. To add more questions, copy the block between the dashed comment lines and paste it inside id="quiz". Update the question number label too.
  6. Optional image: Place your image file in the same folder as this HTML file. Then uncomment the <img> line inside the question card and update the src to your filename.
  7. Delete this yellow box when finished.
