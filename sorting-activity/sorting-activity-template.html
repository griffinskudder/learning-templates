SETUP GUIDE (delete this box when finished)

  1. Change the <title> tag and the <h1> heading to your topic.
  2. Edit the two data-name attributes on the .column divs to set your category names (e.g. data-name="Advantages"). The column headers update automatically.
  3. Find the <!-- ITEM POOL --> section below and edit the 8 data-label values on each <div class="card" ...> to your own items.
  4. Add or remove <div class="card" ...> blocks to change the number of items.
  5. Adjust colours in the :root block at the top of the <style> section.
  6. Delete this yellow instructions box before sharing with students.
